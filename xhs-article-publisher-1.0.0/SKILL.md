# 小红书 Article Publisher Skill

## 描述
小红书内容自动化创作工作流。支持从主题询问、热点搜索、文章生成、配图创建到**一键复制输出**的完整流程。

## 触发场景
- 用户提到"小红书创作"、"写小红书笔记"、"发布到小红书"
- 需要自动化小红书内容生产流程

## 流程概览
```
用户提供主题 → 搜索热点 → 生成笔记 → 生成配图 → 输出内容（一键复制）
```

## 详细工作流程

### 步骤1：用户提供主题

用户直接告知创作主题，例如：
- "帮我写一篇上饶游玩打卡的小红书笔记"
- "写一篇好物分享的小红书笔记"

---

### 步骤2：搜索热点信息

**搜索工具**：AutoGLM WebSearch（智能网络搜索）

**搜索策略**：分平台搜索，每次搜索一个平台

**搜索命令**：
```bash
python .opencode/skills/autoglm-websearch/websearch.py "{主题} 小红书 热门 笔记"
python .opencode/skills/autoglm-websearch/websearch.py "{主题} 抖音 热门 最新"
```

**搜索顺序**：
1. 小红书（优先，找爆款笔记）
2. 抖音（短视频热点）

**提取信息**：
- 标题（name）
- 摘要内容（snippet）
- 来源链接（url）

**失败处理**：
1. 重试3次
2. 简化搜索关键词
3. 通知用户手动提供素材

---

### 步骤3：生成笔记

**核心要求**：
1. **结合热点素材**：必须基于步骤2搜索到的热点笔记来写
2. **小红书风格**：种草感、emoji丰富、短句分段、有互动感
3. **合规用语**：严禁违反平台规定的内容

**笔记结构**（小红书优化版）：
```
标题（带emoji，吸引眼球）

开头引入（1-2句话，制造共鸣）✨

要点一：{核心内容}
要点说明 + emoji 💡

要点二：{核心内容}
要点说明 + emoji 🌟

要点三：{核心内容}
要点说明 + emoji ✅

💡 总结/建议
精炼总结 + 互动引导

#话题标签1 #话题标签2 #话题标签3
```

**风格要点**：
- ✅ emoji 丰富但不过度（每段1-3个）
- ✅ 短句、短段（1-3句一段）
- ✅ 种草感、分享感
- ✅ 标题带数字、疑问、反差
- ❌ 长段落、长句
- ❌ 学术化、说教
- ❌ 敏感词/违规用语

**标题模板**：
- `震惊！{主题}竟然{结果}😱`
- `{数字}个{主题}小技巧，第{数字}个绝了！`
- `{主题}避坑指南❌，看完省下{金额}！`
- `终于有人把{主题}说清楚了！`

---

### 步骤4：生成配图

**配图工具**：
- 主方案：autoglm-generate-image（AI生成）
- 备用方案：autoglm-search-image（搜索图片）

**小红书图片规格**：
- **封面图**：竖版 3:4（推荐 1080x1440）
- **内容图**：根据内容需要 0-4张
- **总数**：1-5张

**生成命令**：
```bash
python .opencode/skills/autoglm-generate-image/generate-image.py "{图片描述}"
```

**配图描述模板**：
```
封面图：{主题}概念图，{场景描述}，竖版构图，无文字，ins风格，高质量，3:4比例
内容图：{内容要点}展示图，{视觉元素}，无文字，ins风格，高质量
```

**下载并保存配图**：
```powershell
Invoke-WebRequest -Uri "{图片URL}" -OutFile "articles/{日期}-{主题}/images/cover.jpg"
```

### 步骤5：输出内容给用户（⭐ 核心步骤）

> ⚠️ 小红书目前没有开放发布API，创作完成后**直接输出格式化内容**，用户一键复制使用。

**输出模板**：

```
## ✅ 小红书笔记创作完成！

---

### 📝 标题
{标题内容}

---

### 📄 正文（可直接复制）
{格式化好的正文内容，带emoji，分段清晰}

---

### 🏷️ 话题标签
{话题标签列表}

---

### 🖼️ 配图
| 图片 | 路径 |
|------|------|
| 封面图 | `{本地路径}` |
| 内容图1 | `{本地路径}` |
| 内容图2 | `{本地路径}` |

---

### 📱 发布步骤
1. 打开小红书创作者平台或APP
2. 点击「发布笔记」
3. 上传配图
4. 复制标题和正文粘贴
5. 添加话题标签
6. 发布 ✅
```

**输出要求**：
1. **标题单独一行**：方便用户直接复制
2. **正文排版清晰**：段落分明，emoji位置合理，用户无需修改即可使用
3. **话题标签单独列出**：方便用户添加
4. **配图路径明确**：用户知道从哪里取图
5. **发布步骤简洁**：3-5步完成

## 异常处理

### 搜索失败
1. 重试3次
2. 切换搜索源
3. 通知用户手动提供素材

### 图片生成失败
1. 重试3次
2. 降低提示词复杂度
3. 使用搜索图片作为备用

## 注意事项

1. **内容合规**：笔记需符合小红书社区规范

3. **一键复制**：输出格式要便于用户直接使用，无需额外编辑
4. **话题标签**：选择合适的话题标签可以增加曝光
