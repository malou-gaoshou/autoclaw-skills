# 小红书 Article Publisher Skill

小红书内容自动化创作工作流。

## 快速开始

### 1. 开始使用
```
用户: 帮我写一篇小红书笔记，主题是{主题}
Agent: [自动执行完整流程]
```

## 工作流程

```
询问主题 → 搜索热点 → 生成笔记 → 生成配图 → 导出内容
```

### 步骤说明

| 步骤 | 工具 | 说明 |
|------|------|------|
| 1. 询问主题 | - | 9:00 定时询问今日创作主题 |
| 2. 搜索热点 | autoglm-websearch | 搜索小红书/抖音/公众号热点 |
| 3. 生成笔记 | blog-writer | 结合热点，生成小红书风格笔记 |
| 4. 生成配图 | autoglm-generate-image / autoglm-search-image | 封面图竖版3:4，内容图多宫格 |
| 5. 导出内容 | - | 导出内容供手动发布 |

## 小红书特点

### 笔记风格
- emoji 丰富但不过度
- 短句、短段
- 种草感、分享感
- 标题带数字、疑问、反差

### 图片规格
- 封面图：竖版 3:4（1080x1440）
- 内容图：可横可竖，最多9张

### 发布方式
- 小红书无开放API
- 需要手动复制内容到小红书APP发布

## 依赖技能

| 技能 | 用途 |
|------|------|
| autoglm-websearch | 网络搜索热点信息 |
| blog-writer | 生成笔记内容 |
| autoglm-generate-image | AI 生成配图 |
| autoglm-search-image | 搜索配图（备用） |

## 文件说明

| 文件 | 说明 |
|------|------|
| SKILL.md | 主文档，完整工作流程 |
| article_template.md | 笔记结构参考 |

## 注意事项

1. 封面图必须是竖版3:4
2. 笔记需符合小红书社区规范
3. 需要手动发布到小红书APP
4. 选择合适的话题标签增加曝光
